### Quickstart
    $ brew install redis
    $ npm install
    $ redis-server
    $ node app
    
### Dependencies

- Node.js (the server)
- Express.js (lightweight framework on top of node)
- Redis (simple key-value data store)
- Angular.js (for client side JS)
- Jade (HTML rendering engine)

### Develop locally

Start the redis server

    $ redis-server

Start the node app

    $ node app
